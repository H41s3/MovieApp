# BMovie

A SwiftUI iOS app for browsing movies, with tabs for Home, Upcoming, and Search.

## Requirements

- Xcode 15+
- iOS 17+
- Swift 5.9+

## Getting Started

1. Clone the repo
2. Open `BMovie/BMovie.xcodeproj` in Xcode
3. Select a simulator or device and run

## Project Structure

```
BMovie/
├── BMovie.xcodeproj/       # Xcode project file
└── BMovie/
    ├── BMovieApp.swift     # App entry point
    ├── ContentView.swift   # Main tab view
    └── Assets.xcassets/    # App icons and colors
```
